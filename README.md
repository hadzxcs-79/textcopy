# textcopy
Last updated: July 2026

## Overview

This Chrome extension allows users to save text that they explicitly select on web pages.

## Information We Collect

The extension stores only the text that the user intentionally chooses to save.

No personal information is collected automatically.

The extension does not collect:

- Names
- Email addresses
- Passwords
- Payment information
- Browsing history
- Location information
- Keystrokes
- Mouse movements

## Data Storage

Saved text is stored locally using Chrome Storage.

No data is transmitted to external servers.

## Third-Party Sharing

The extension does not sell, share, or transfer user data to any third party.

## User Control

Users can delete all saved content at any time through the extension.
